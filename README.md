#RECLAIM - Cognitive Engagement Monitor
Samvidhi 2026 Hackathon Rapid Prototype

## Overview
A Python-based engagement monitoring system developed during a 24-hour hackathon.

## Features
- Face detection
- Eye tracking
- Activity monitoring
- Voice alerts
- GUI notifications

## Technologies
- Python
- OpenCV
- Tkinter
- SpeechRecognition
- pyttsx3

## Hackathon
Developed a cognitive engagement monitoring prototype using Python, OpenCV, speech recognition, and GUI alerts during a 24 hour hackathon.
AI-assisted development tools were used to accelerate implementation and experimentation during rapid prototyping.
